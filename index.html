Introduction SELECT WHERE JOIN GROUP BY

1. What is SQL?

SQL (Structured Query Language) is used to communicate with databases. It allows you to store, retrieve, update and delete data.

SQL is used in databases like:

  • MySQL
  • PostgreSQL
  • SQL Server
  • Oracle

2. SELECT Statement

The SELECT statement is used to retrieve data from a database.

SELECT * FROM employees;

Select specific columns:

SELECT name, salary FROM employees;

3. WHERE Clause

WHERE filters records.

SELECT * FROM employees WHERE salary > 50000;

Multiple conditions:

SELECT * FROM employees WHERE department = 'IT' AND salary > 60000;

4. JOIN

JOIN combines rows from multiple tables.

SELECT employees.name, departments.department_name FROM employees JOIN departments ON employees.department_id = departments.id;

5. GROUP BY

GROUP BY groups rows with the same values.

SELECT department, COUNT(*) FROM employees GROUP BY department;
